# India's Agricultural Crop Production Analysis

This project provides a comprehensive data analysis of crop production trends across India.

## Project Overview
This repository contains the documentation and visualizations for my analysis, covering trends from 1997–2020.

## Dashboard Components
- **Time-Based Analysis**: Sheets visualizing production fluctuations over the years.
- **Regional Comparison**: Interactive maps and bar charts comparing state-level agricultural output.
- **Interactive Dashboards**: A collection of 7 dashboards allowing users to filter data by crop, state, and year.
